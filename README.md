Hospital web platform designed for the
**Hospital Regional Traumatológico y Quirúrgico Prof. Juan Bosch**
in La Vega, Dominican Republic.

Built to automate medical appointment workflows, improve scheduling reliability, and modernize administrative operations.

### Highlights

* JWT authentication with bcrypt
* RBAC authorization system
* Immutable audit logging
* Transaction-safe scheduling
* Partial unique indexes for concurrency protection
* TIMESTAMPTZ timezone correctness
* CI/CD pipeline on every commit
* Full Docker containerization
* Automated testing suite

<br/>
